Úvod

Cascading Style Sheets (CSS), česky kaskádové styly, je stylový jazyk sloužící pro popis vzhledu a formátování dokumentů napsaných nějakým značkovým jazykem. Odděluje styl dokumentu od jeho obsahu. Nejčastěji se používá pro přidávání stylů (např. fonty, barvy, pozadí, ohraničení, formátování textu) do webových dokumentů. Kaskádové styly umožňují definovat způsob zobratení každého elementu na stránce. CSS lze propojit s HTML dokumentem jako vnitřní, externí nebo vložené styly.

Možnosti propojení s HTML dokumentem

Vnitřní styly

Metoda propojení prostřednictvím vnitřních stylů spočívá v uvedení CSS kódu do hlavičky HTML souboru, který chceme stylovat pomocí CSS. Každý HTML soubor tak bude obsahovat CSS ke stylování dané stránky. Tato metoda je vhodná jen pokud chceme stylovat jedním stylem pouze jednu stránky. Kdybychom totiž chtěli provést jednu a tu samou změnu na více stránkách, museli bychom změnit CSS kód v každém HTML zvlášť
